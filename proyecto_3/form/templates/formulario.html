Validaciones JS

Los campos se validarán de la siguiente manera

  • Ficha:

    se puede ingresar cualquier caracter

  • Nombre:

    solo se puede ingresar letras

  • Apellidos:

    solo se puede ingresar letras

  • Correo:

    solo se puede ingresar Correo válido

  • Teléfono:

    solo se puede ingresar números

  • Género:

    solo se puede seleccionar un Género

  • Ciudad:

    solo se puede seleccionar una Ciudad

  • Habilidades:

    solo se puede seleccionar como mínimo 3 Habilidades
